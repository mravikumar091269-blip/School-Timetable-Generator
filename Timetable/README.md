# 📅 Timetable Generator

A web-based Timetable Generator built using  **HTML**, **CSS**, and **JavaScript**. The application generates a class timetable based on user inputs such as standard, section, subjects, the required number of periods and teacher details.

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

## 🚀 How to Run

1. Clone or download the repository.
2. Open the project folder.
3. Open `new.html` in your web browser.
